# 🧠 Imersão Dev – Alura (9ª Edição)

Este repositório reúne os mini projetos desenvolvidos durante a **Imersão Dev da Alura – 9ª Edição (2025)**. A cada aula foi proposto um desafio prático com foco em **HTML, CSS e JavaScript**, explorando criatividade, lógica de programação e manipulação da interface.

## 📁 Estrutura do Repositório

Organizado por aulas, com um projeto principal em cada:

- 📂 **Aula 01 – Conversor de Moedas**
  - Projeto que converte valores em real para outras moedas como dólar, usando cálculos simples e interface responsiva.

- 📂 **Aula 02 – Jokenpô**
  - Jogo de pedra, papel e tesoura com interação entre usuário e computador.

- 📂 **Aula 03 – Round 6**
  - Jogo simples inspirado na série "Round 6", com lógica de eliminação e mensagens dinâmicas.

- 📂 **Aula 04 – Heróis vs Vilões**
  - Sistema que sorteia personagens aleatórios (heróis ou vilões) e exibe o vencedor com base em atributos.

- 📂 **Aula 05 – Quiz Pokémon**
  - Quiz de múltipla escolha com pontuação e verificação de respostas corretas ao final.

## 🚀 Tecnologias Utilizadas
- HTML5
- CSS3
- JavaScript
